# Rodar o Projeto
Para executar o projeto, clone o repositório no VSCode, abra o terminal e execute os comandos (um de cada vez):
```
git clone https://github.com/filipedeschamps/video-maker.git
cd desafio-dev-venturelabs
npm install
npm start
```
